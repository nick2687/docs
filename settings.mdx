# Settings

The settings section allows you to configure all aspects of your HomeLens account, from profile information to integrations and branding.

## Settings Overview

Settings are organized into several sections:

- **Profile**: Personal and account information
- **Company**: Business details and branding
- **Services**: Inspection services and pricing
- **Templates**: Inspection templates
- **Appearance**: Visual customization
- **Integrations**: Third-party connections
- **Portal**: Client portal configuration
- **Widget**: Booking widget settings
- **Payments**: Payment processing
- **Communication Templates**: Email and SMS templates
- **Password**: Change password

## Profile Settings

### Personal Information

Update your profile:

- **Email**: Account email (managed by Firebase Auth)
- **Phone Number**: Business phone
- **Profile Photo**: Upload profile picture

### Account Information

- **License Number**: Inspector license number
- **Company Name**: Business name
- **Business Address**: Company address
- **Website**: Company website URL

## Company Settings

### Business Details

- Company name
- Business address
- Phone number
- Website
- License information

### Branding

Customize your brand appearance:

- **Logo**: Upload company logo
- **Colors**: 
  - Primary color
  - Secondary color
  - Accent color
  - Sidebar color
  - Button color
- **Font**: Custom font family

## Service Settings

Manage inspection services:

- Create new services
- Edit existing services
- Configure pricing
- Set default service
- Activate/deactivate services

See [Services](/docs/services) for detailed documentation.

## Template Settings

Manage inspection templates:

- Create templates
- Edit templates
- Import/export templates
- Organize sections and items

See [Templates](/docs/templates) for detailed documentation.

## Appearance Settings

Customize application appearance:

### Theme

- **Light Mode**: Default light theme
- **Dark Mode**: Dark theme option
- **System**: Follow system preference

### Dashboard

- Sidebar behavior
- Layout preferences
- Display options

## Integration Settings

Configure third-party integrations:

- **Stripe**: Payment processing
- **Google Calendar**: Calendar sync
- **Google Drive**: Report storage
- **HelloSign**: E-signatures
- **HomeBinder**: Report platform
- **ISN**: Order management

See [Integrations](/docs/integrations) for detailed documentation.

## Portal Settings

Configure client portal:

- **Welcome Message**: Custom message for clients
- **Access Expiration**: Days until access expires
- **Payment Options**: Enable payment collection
- **Closing Date**: Show option period dates
- **Branding**: Portal appearance

See [Client Portal](/docs/client-portal) for detailed documentation.

## Widget Settings

Configure booking widgets:

- **General**: Basic widget settings
- **Branding**: Widget appearance
- **Fields**: Custom fields
- **Scheduling**: Availability rules
- **Payment**: Payment options
- **Embed Code**: Get embed code

See [Widgets](/docs/widgets) for detailed documentation.

## Payment Settings

Configure payment processing:

- **Stripe Connect**: Connect Stripe account
- **Account Status**: View connection status
- **Payment Methods**: Supported methods
- **Invoice Settings**: Invoice configuration

## Communication Templates

Manage email and SMS templates:

- **Email Templates**: Create email templates
- **SMS Templates**: Create SMS templates
- **Variables**: Use dynamic variables
- **Preview**: Test templates

## Password Settings

Change your password:

1. Enter current password
2. Enter new password
3. Confirm new password
4. Save changes

**Security Requirements:**

- Minimum 8 characters
- Include uppercase and lowercase
- Include numbers
- Include special characters (recommended)

## Sync Settings

Configure data synchronization:

- **Mobile Sync**: Sync with mobile apps
- **Backup**: Automatic backups
- **Export**: Export data

## Best Practices

### Organization

- Keep settings organized
- Review settings periodically
- Update branding as needed
- Test changes before deploying

### Security

- Use strong passwords
- Enable two-factor authentication (if available)
- Review integration permissions
- Monitor account activity

### Branding

- Use consistent branding
- Optimize logo size
- Test colors for accessibility
- Ensure mobile compatibility
